# Aure Relics Local Fonts

This folder is reserved for local font files so Aure Relics Digital Tabletop Grid keeps the same visual style when played offline.

The CSS will look for these files:

```text
Cinzel-VariableFont_wght.woff2
Cinzel-VariableFont_wght.ttf
Spectral-Regular.woff2
Spectral-Regular.ttf
Spectral-SemiBold.woff2
Spectral-SemiBold.ttf
Spectral-Bold.woff2
Spectral-Bold.ttf
```

You do not need both `.woff2` and `.ttf`; the CSS lists both so the browser can use whichever format is present.

Recommended source: download Cinzel and Spectral from the official Google Fonts pages and keep the license files that come with the font downloads.

Do not remove this folder from offline release packages.
