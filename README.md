# Aure Relics Battle Grid

A simple digital tabletop grid for Dungeons & Dragons.

Designed for use on a TV or shared screen to visually track combat, terrain, and positioning.

---

## Features

- Grid-based battle map
- Click-and-drag wall placement
- Terrain tools (trees, rocks, fire, doors, water, etc.)
- Token system with auto-numbering (P1, E1, N1, B1)
- Scene saving and loading
- Brush size (1x1, 2x2, 3x3)

---

## How to Use

1. Open the app in your browser
2. Select a tool from the sidebar
3. Click or drag on the grid to build your map
4. Use token buttons to place players and enemies
5. Save scenes to switch between encounters

---

## Notes

- Each user’s saved scenes are stored locally in their browser
- No multiplayer support yet (planned for future versions)
